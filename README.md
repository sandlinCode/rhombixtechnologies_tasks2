# 🎀 Cute Hangman Game (PyQt5)

A fun, pastel-themed **Hangman Game** built using **Python (PyQt5 GUI)**.  
Players can guess letters with on-screen buttons, see their progress live, and restart anytime — all in a cute, modern interface 💖.

---

## 🧩 Features

✨ Full PyQt5 graphical interface  
🎨 Cute soft-pink theme and rounded buttons  
🔠 Alphabet buttons (A–Z) for guessing  
❤️ Attempts counter with popups for Win / Lose  
🔁 Restart button to play again  
🧠 Works fully offline  

---



## 🧰 Requirements

- Python 3.8+
- PyQt5

pip install PyQt5
